How they hold together

Gates  Stop hook

educate and research each ship a Stop hook that refuses to finish while a status runs ahead of the artifacts that prove it.

If a skill "won't let you finish," it's naming a missing artifact — that's the guardrail working, not a bug.

Handoffs  shared transport

Plugins compose only through files: a gitignored .handoff/ transport carries requests, with evidence kept in tracked state.

A lesson emits a SPEC → build consumes it → build returns findings. No plugin calls another directly.

Restart Claude Code — these installed after the current session started. Update flow: node scripts/build.mjs → claude plugin marketplace update praxis → reinstall. Source: educate@praxis · research@praxis · build@praxis
